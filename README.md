# 🎨 Gainhub - Skill Exchange Platform

![Gainhub Banner](https://images.unsplash.com/photo-1522202176988-66273c2fd55f?auto=format&fit=crop&q=80&w=1200)

## 🌟 Overview

Gainhub is a modern skill exchange platform that connects people who want to share their expertise with those eager to learn.

## 🎯 Features

- ✅ User Authentication (Sign up & Login)
- ✅ Profile Management
- ✅ Skill Exploration
- ✅ Connection Requests
- ✅ Bilingual Support (English / العربية)
- ✅ 3D Tilt Effects on Cards
- ✅ Fully Responsive Design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Fonts

## 📁 Files

- `index.html` - Main structure
- `SkillFrontEnd.css` - Styling
- `script.js` - Functionality

## 🚀 How to Use

1. Download all three files
2. Save them in the same folder
3. Open `index.html` in your browser
4. Sign up and start exchanging skills!

## 🌍 Language

Click the 🌐 button to switch between English and Arabic.

## 📱 Responsive

Works on mobile, tablet, and desktop.

## 🔧 Customization
